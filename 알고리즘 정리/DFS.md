# DFS(Depth First Search, 깊이 우선 탐색)

## 깊게 들어갔다가 더 이상 갈 곳이 없으면 뒤로 돌아옴

> 방식
>- 가능한 깊이 끝까지 파고 들어간 후 돌아오는 방식
-------------------------------------------
> 구현
>- 재귀 함수
>- 직접 스택 사용
-------------------------------------------
> 특징
>- 경로 탐색에 유리
>- 백트래킹 문제(미로 찾기, 조합 탐색)에 활용
---------------------------------------------
> 예시
>- 그래프에서 특정 노드까지 가는 경로가 있는지 확인
>- 순열/조합 생성

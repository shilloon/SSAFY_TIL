# 그래프(Graph)

## 관계를 표현하는 자료구조

> 정의
>- 정점(Vertex, 노드)과 간선(Edge)로 이루어진 구조
-------------------------------------------------
> 종류
>- 방향 그래프(유향) / 무방향 그래프(무향)
>- 가중치 그래프(간선마다 비용/거리 있음)
--------------------------------------------------
> 표현 방식
>- 인접 행렬 (2차원 배열)
>- 인접 리스트 (리스트 안에 리스트/딕셔너리)
--------------------------------------------------
> 활용
>- 네트워크 연결
>- 최단 경로 (다익스트라, BFS)
>- 위상 정렬
